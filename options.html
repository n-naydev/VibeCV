CV Tailor – Settings

Configure your OpenAI access and base CV once. The extension will use this to tailor PDFs for each job you open.

AI API Required

Your key is stored locally in Chrome storage and is only used to talk directly to OpenAI from your browser.

OpenAI, Gemini or Claude

You can create a key in your OpenAI dashboard. Keep it private.

Any GPT-4.1 / GPT-4.1-mini compatible model name will work.

Base CV Once

You can either upload a PDF CV to extract the text, or paste/edit your base CV manually. This base CV is what the extension tailors for each job.

Use this if you have a PDF CV and don’t want to copy-paste the text.
